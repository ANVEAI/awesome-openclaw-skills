# Contributing to Awesome OpenClaw Skills

Thank you for your interest in contributing to this awesome list! 🎉

## How to Contribute

### Adding a New Skill

1. **Publish your skill to ClawHub first**
   - Your skill must be available on [ClawHub](https://clawhub.ai)
   - Ensure it has a complete SKILL.md with proper documentation

2. **Fork this repository**
   ```bash
   git clone https://github.com/ANVEAI/awesome-openclaw-skills.git
   cd awesome-openclaw-skills
   ```

3. **Add your skill to the appropriate category**
   - Choose the most relevant category
   - Add your skill to the table following the existing format
   - Include: Skill name, description, and install command

4. **Submit a pull request**
   - Provide a clear description of your skill
   - Include a link to your ClawHub page
   - Mention any unique features or use cases

### Skill Requirements

To be included in this list, your skill must:

- ✅ Be published on ClawHub
- ✅ Have a comprehensive SKILL.md
- ✅ Include installation instructions
- ✅ Be actively maintained
- ✅ Pass basic security checks
- ✅ Have a clear, descriptive name
- ✅ Include a valid GitHub repository link

### Categories

Choose the most appropriate category for your skill:

- **Voice AI & Communication** — Voice assistants, transcription, audio processing
- **Customer Support** — Helpdesk, ticketing, support automation
- **Marketing & Sales** — CRM, email campaigns, lead generation
- **Productivity & Tasks** — Task management, scheduling, time tracking
- **Analytics & Intelligence** — Data analysis, reporting, business intelligence
- **Coding Agents & IDEs** — Development tools, code review, testing
- **Web & Browser Automation** — Web scraping, browser automation, APIs
- **Communication Tools** — Messaging, notifications, team collaboration
- **Security & Monitoring** — Security scanning, uptime monitoring, logging

### Formatting Guidelines

Use this format when adding a skill:

```markdown
| [skill-name](https://github.com/username/repo) | Brief description of what the skill does. | `clawhub install skill-name` |
```

### Featured Skills

To be considered for the **Featured** section at the top, your skill should:

- Have significant community adoption
- Be well-documented with examples
- Solve a unique or important problem
- Be actively maintained for 3+ months
- Pass security audits

### Updating Existing Skills

If you need to update information about an existing skill:

1. Fork the repository
2. Make your changes
3. Submit a pull request with a clear description

### Removing Skills

Skills may be removed if they:

- Are no longer maintained
- Have security vulnerabilities
- Violate OpenClaw terms of service
- Are duplicates of existing skills

## Code of Conduct

- Be respectful and constructive
- Focus on quality over quantity
- Help others learn and improve
- Follow OpenClaw community guidelines
